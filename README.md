## Sobre o projeto :computer:

Decidi criar, pela primeira vez, um painel administrativo fictício que contém diversos elementos como gráficos, cards e tabelas. 
Utilizei a lib Recharts para criar gráficos interativos e responsivos; ReactJS com NextJS para otimização e performance e TailwindCSS
para estilização.

---

## Tecnologias usadas :star:

- [x] NextJS
- [x] ReactJS
- [x] TailwindCSS
- [x] TypeScript
- [x] Recharts
